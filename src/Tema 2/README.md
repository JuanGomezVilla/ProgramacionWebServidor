## Creación
1. Crear el proyecto, instalar *chalk* para la consola:
    ```bash
    mkdir practica-1
    cd practica-1
    npm init -y
    npm install chalk@4
    ```
2. Crear dos archivos:
    - ***app.js*:** contiene el código a ejecutar desde la consola
    - ***gestorEventos.js*:** código que gestiona los eventos, importado por *app.js*
