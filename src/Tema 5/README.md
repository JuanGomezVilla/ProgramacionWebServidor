# Tema 5

## Primeros pasos
1. Iniciar el proyecto:
    ```bash
    npm init -y
    npm install express
    npm install express-validator
    npm install jsonwebtoken
    ```
2. Generar estructura:
    ```bash
    mkdir config
    mkdir controllers
    mkdir middlewares
    mkdir models
    mkdir routes
    mkdir services
    mkdir tests
    echo > app.js
    ```
3. Ejecutar:
    ```bash
    node app.js
    node --watch app.js
    ```
